🚢 Kochi Water Metro Rail Project

📌 About the Project

The Kochi Water Metro Rail Project is a smart transportation system inspired by the real-world Kochi Water Metro in Kochi.

This project demonstrates how a water-based metro system can be digitally managed using modern technologies. It focuses on route management, scheduling, digital ticket booking, and sustainable transportation modeling.

🎯 Objective

Reduce road congestion

Promote eco-friendly electric ferry transport

Provide smart scheduling and ticketing system

Improve passenger experience through digital solutions
